# Soundbender TagDeck 1.4.0-beta.1 - Beta 1

First public/tester beta.

## Included

- Windows installer.
- Source code package for inspection, audit and forks.
- Local-first music library.
- Integrated audio player.
- Safe metadata reading and editing.
- Internal creative organization.
- Explorer / Curator mode.
- Workflow presets.
- Session mode.
- Playlists.
- CSV/JSON export and import.
- Full library backup/restore foundation.
- Manual update checker using soundbender.live.

## Safety

- TagDeck does not upload your music.
- TagDeck does not require accounts.
- Internal creative data stays in SQLite.
- Removing songs from TagDeck does not delete audio files.
- Emptying the library does not delete audio files.
- File metadata writes use backup and validation.

## Known Limitations

- This is beta software.
- Automatic self-updating is not enabled yet.
- Mobile companion app is future roadmap, not included.
- Cloud sync is not included.
- GitHub publishing is not active yet; this beta source package is shared
  through Google Drive.
