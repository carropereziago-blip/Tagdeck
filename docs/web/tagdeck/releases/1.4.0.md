# Soundbender TagDeck 1.4.0 - Public launch build

## English

This is the first public launch build prepared for manual distribution.

- Added customizable keyboard shortcuts for Library, Explorer and Session.
- Added guided shortcut value/action controls so Rating, Status, Action and
  Language use valid internal values.
- Added reorderable Library columns.
- Added ordered Library selection handoff into Explorer and Session queues.
- Includes the latest Session, Explorer, export/import and updater hotfixes.
- Update manifest now uses `latest.txt` for Porkbun/static hosting compatibility.
- Refreshed Windows installer, update manifest and manual GitHub/source package.

## Espanol

Esta es la primera build de lanzamiento publico preparada para distribucion
manual.

- Atajos de teclado personalizables para Biblioteca, Explorador y Sesion.
- Controles guiados de valor/accion para que Rating, Estado, Accion e Idioma
  usen valores internos validos.
- Columnas reordenables en Biblioteca.
- Envio ordenado de seleccion de Biblioteca a colas temporales de Explorer y
  Sesion.
- Incluye los ultimos hotfixes de Sesion, Explorer, export/import y
  actualizador.
- El manifest de actualizacion ahora usa `latest.txt` para compatibilidad con
  Porkbun/hosting estatico.
- Instalador Windows, manifest de actualizacion y paquete GitHub/codigo fuente
  actualizados.
