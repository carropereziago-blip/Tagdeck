# Soundbender TagDeck 1.4.0-beta.2 - Beta 2 hotfix

## English

This hotfix refreshes the beta installer and source package after the latest
Session Mode and Explorer fixes.

- Session Mode no longer gets stuck on "Preparing session...".
- Loading a playlist in Session now keeps the remaining queue visible and ready.
- Playing a song manually from Session search/suggestions no longer clears the queue.
- Explorer Criterion no longer shows duplicate "All" entries.
- The update manifest has been refreshed for manual update checks.

## Espanol

Este hotfix actualiza el instalador beta y el paquete de codigo fuente tras las
ultimas correcciones de Modo Sesion y Explorer.

- Modo Sesion ya no se queda bloqueado en "Preparando sesion...".
- Al cargar una lista en Sesion se mantiene la cola restante visible y lista.
- Reproducir una cancion manualmente desde busqueda/sugerencias de Sesion ya no borra la cola.
- El criterio de Explorer ya no muestra "All" duplicado.
- El manifest de actualizacion queda actualizado para el comprobador manual.
