# Product Vision

Soundbender TagDeck helps AI music creators turn generated-song chaos into a
local, reviewable, organized, release-ready library.

## Problem

AI music tools make it easy to create more songs than a person can remember.
After a few sessions, creators often have hundreds or thousands of files:

- multiple versions of the same idea,
- unfinished takes,
- hidden strong parts,
- tracks that need DAW work,
- radio candidates,
- release candidates,
- archives that should not distract from active work.

Normal file browsers and media players do not capture the creative decisions
around those songs.

## Core Loop

TagDeck is built around a fast review loop:

```text
listen -> rate -> decide -> tag -> save and next
```

The goal is to make decisions while listening, not after losing track of the
library again.

## Workflows

TagDeck supports creative workflows such as:

- DAW finishing,
- radio selection,
- release preparation,
- archive cleanup,
- model-seed collection,
- playlist building,
- version comparison.

## Positioning

TagDeck is not a DAW.

TagDeck is not a cloud platform.

TagDeck is a local curation workstation for generated music libraries.
