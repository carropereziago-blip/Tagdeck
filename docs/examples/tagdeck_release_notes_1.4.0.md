# Soundbender TagDeck 1.4.0

## English

Public launch build for manual distribution.

- Customizable keyboard shortcuts for Library, Explorer and Session.
- Guided shortcut values/actions with valid internal values.
- Reorderable Library columns.
- Ordered Library selection handoff into Explorer and Session.
- Latest Session, Explorer, export/import and update hotfixes.
- Update manifest now uses `latest.txt` for Porkbun/static hosting compatibility.
